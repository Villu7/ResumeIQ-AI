# ResumeIQ-AI

An AI-powered Resume Analyzer and ATS Scoring platform built using React and modern web technologies.

## Features

- AI-powered ATS Resume Scoring
- Resume Upload and Analysis
- AI Interview Question Generator
- Responsive Modern UI
- Resume Tracking Dashboard
- Skill-Based Feedback System

## Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- Vite
- React Router
- Zustand

## Installation

```bash
npm install
npm run dev
```

## Project Goal

This project helps users analyze resumes, improve ATS compatibility, and prepare for interviews using AI-powered feedback and smart evaluation techniques.
